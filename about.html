About Smart Exam Invigilation System

This project is a modern, secure, and user-friendly platform for conducting online exams with live proctoring and suspicion tracking. It is designed for college-level use and demonstrates practical applications of data structures, algorithms, and real-time monitoring.

Key Features

  • Admin Portal: Create exams, add questions, and review suspicion logs.
  • Student Portal: Take exams, answer shuffled questions, and receive instant results.
  • Suspicion Tracking: Detects tab switch, inactivity, right-click, copy/paste, and live face detection (no face, multiple faces).
  • Suspicion Logs: Admins can review all suspicious activity for every exam attempt.
  • Modern UI: Clean, responsive design for both admin and student interfaces.

Technologies Used

  • Python (Flask) for backend
  • HTML, CSS, JavaScript for frontend
  • MediaPipe Face Detection (classic JS API)
